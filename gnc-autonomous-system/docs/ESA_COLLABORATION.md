# ESA Collaboration Context

## Research Background

This project was developed as part of a Master's thesis research collaboration between:
- **Politecnico di Torino** (Italy)
- **ISAE-SUPAERO** Space Advanced Concepts Laboratory - SaCLaB (France)
- **European Space Agency (ESA)** stakeholders

**Title**: "Systems Engineering for Asteroid Exploration GNC Validation Test"

**Academic Year**: 2022-2023

## Research Objectives

The research aimed to develop a comprehensive methodology for:

1. **GNC System Validation** for autonomous asteroid exploration missions
2. **MBSE Application** using SysML for space GNC systems
3. **Requirements Traceability** following ECSS standards
4. **Hardware-in-the-Loop Testing** strategy development

## ESA Context

This work supports ESA's ongoing efforts in:
- Autonomous spacecraft operations for deep space missions
- Small body exploration mission preparation
- GNC technology development and validation
- MBSE adoption in space systems engineering

## Key Contributions

### 1. Functional Analysis (Chapter 4)
- Mission objectives tree following MBSE principles
- 50+ GNC requirements with full traceability
- Functional decomposition using SysML
- Activity diagrams for mission phases

### 2. System Architecture (Chapter 5)
- Complete GNC system architecture design
- Sensor suite definition and trade-offs
- Control algorithms specification
- Interface definition following ECSS standards

### 3. Verification Strategy (Chapter 6)
- V&V methodology aligned with ECSS-E-ST-60-30C
- Hardware-in-the-loop testing approach
- Requirements verification matrix
- Test case definitions

## Heritage Missions

The work builds on lessons learned from:
- **Hayabusa** (JAXA, 2003-2010)
- **Hayabusa2** (JAXA, 2014-2020)
- **OSIRIS-REx** (NASA, 2016-2023)
- **Hera** (ESA, upcoming)

## Methodology

**Model-Based Systems Engineering (MBSE)**:
- Tool: CATIA Cameo Systems Modeler
- Language: SysML (Systems Modeling Language)
- Framework: ESA SysML Solution guidelines
- Standards: ECSS-E-ST-10C, ECSS-E-ST-60-30C
